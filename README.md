# Анализ дефектов металлических поверхностей методом глубокой кластеризации

Реализация модифицированной модели VaDE с энтропийной регуляризацией для безметочной сегментации промышленных дефектов.

## Особенности
- Использует датасет [Metal Surfaces Defects](https://www.kaggle.com/datasets/toqaalaaawad/metal-surfaces-defects) (12 классов)
- Показан компромисс между качеством (NMI) и интерпретируемостью (энтропия)
- Оптимальное значение η = 0.2 для задач промышленной безопасности

## Запуск
1. Откройте `metal_defects_experiment.ipynb` в [Google Colab](https://colab.research.google.com/)
2. Загрузите `kaggle.json` для доступа к данным
3. Запустите все ячейки
